## Description

디프만 10기 6조 Backend Server Repository

## Install Node

```bash
# This project uses the node14 version.
# Please install node14 version

# Window
$ choco install nodejs-lts

# Mac
$ brew install node@14

# Linux
$ curl -fsSL https://deb.nodesource.com/setup_14.x | sudo -E bash -
$ sudo apt-get install -y nodejs
```

## Running the app

```bash
# install node Module
$ npm install

# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
